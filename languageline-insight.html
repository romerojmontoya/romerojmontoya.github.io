Overview

This guide walks IT support staff through installing and activating the LanguageLine InSight app on a store iPad. Once activated, the device gives store associates on-demand access to live video and audio interpreters in 240+ languages.

Who performs this task

IT Support — Tier I

When to use this guide

New store · Device installation · Device replacement

Devices affected

Language / Translation iPads

Outcome

LanguageLine activated and ready for store use

Prerequisites

Have the following before performing installation:

Latest version of iOS is installed
IT Apple account is signed in
Store number is known (e.g., 029)
iPad has an active Wi-Fi connection
ℹ

Use IT's Apple ID

Store iPads are managed under a shared Apple ID. You do not need individual credentials to install the app.

1

Install the app

1

Unlock the iPad.

Enter PIN [PIN REDACTED] on the lock screen.

2

Open the App Store.

Tap the App Store icon on the home screen. If the icon is not visible, swipe right to open App Library and search for "App Store".

3

Search for LanguageLine InSight.

Tap the search bar and type LanguageLine InSight. The publisher is [App Publisher] — confirm this before installing to avoid a similarly named app.

4

Tap Get to begin the download.

Wait for installation to complete. The button will change to Open when ready.

5

Return to the home screen.

Press the Home button or swipe up from the bottom. Locate the LanguageLine InSight icon on the home screen or in App Library.

2

Activate the device

Activation links this iPad to the correct store account in the LanguageLine system. Use the credentials below.

Activation credentials
Authentication code [AUTH CODE REDACTED]
Device name format
0[Store#]
e.g., Store 29 → 029
1

Open the LanguageLine InSight app.

The app opens to an activation screen with two text fields and a greyed-out Activate Device button.

2

Enter the authentication code.

In the first field, enter [AUTH CODE REDACTED] exactly as shown. The code is case-sensitive.

3

Enter the device name.

In the second field, enter the store number using the format 0[Store#]. Use leading zeros — for example, Store 29 → enter 029.

4

Tap Activate Device.

The button becomes active once both fields are filled. After tapping, the app will prompt the store to select a language — the device is now ready for use.

⚠

Device name cannot be changed after activation

Double-check the store number before tapping Activate Device. If the wrong name is entered, contact IT to reset the device registration.

3

Verify setup

Confirm the device is functioning correctly before handing off to the store.

1

Confirm the language selection screen appears.

Immediately after activation, the app should display a list of available languages. This confirms the device has connected to the LanguageLine network.

2

Select Spanish and initiate a test call.

Tap Spanish from the language list, then tap Video to request an interpreter. Confirm the call connects. End the call immediately after confirming — do not wait for an interpreter.

✓

Setup complete

If the language list appeared and the test call initiated, the device is configured correctly. Inform the store that the iPad is ready for interpreter sessions.

Troubleshooting

"Activate Device" button remains greyed out

Both fields must be filled before the button activates. Tap each field to confirm the keyboard appears and the value was saved.

App is not visible in the App Store search results

Ensure the iPad is connected to Wi-Fi. If confirmed, sign out of the App Store and sign back in, then search again for LanguageLine InSight.

Activation fails with an error message

Verify the code is entered exactly as [AUTH CODE REDACTED] with no spaces. If the error persists, contact the IT helpdesk — the code may have been rotated.
